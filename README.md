# FFMpeg-Manual-Edit-Scripts
Video editing done manually on 100% GPU side with FFMPEG :ghost: maximum speed video editing :bicyclist:

I done video editing before for some videos but i hate you need strong CPU for video editing on Windows.My only two options was either 100% crank up my CPU or install linux and use GPU encodig. Video editing on Windows is very limited on GPU (not free as well) so this is the third alternative.

> [!NOTE]
This is a zero encode process thats why you need at least a decade old CPU (just enough to run) but not so garbage GPU and SSD. So as long as your computer is stronger than a calculator you dont need to wait for hours to edit a video.<

> [!IMPORTANT]
You can run **Download (FFMpeg).cmd** file daily it will check date of your .zip file date and update if its outdated (DAILY)
> 

# You Need
```
Any AMD GPU has Adrenalin Software (Record and Edit)
Any 2010+ CPU or Above (Like FX6300 or Intel I5 Series)
Any SSD (SATA or M.2 Doesnt Matter)
Any Ram Size DDR3+ Above (I Recommend Min 8 GB)
Windows 10 or 11
FFMPEG (Use Latest One with Download CMD File)
```
# Make Sure Use It In Correct Order
> [!IMPORTANT]
Step 1 **Create a Folder**\
Step 2 **Execute Download (FFMpeg).cmd** This will download latest FFmpeg (You can use everyday)\
Step 3 **Record your video with AMD Adrenalin and edit/take clips (HEVC is better)**\
Step 4 **Run Merge (Video).cmd** to merge your clips (If you have 2 seperate videos) after you cut them with AMD Adrenalin. This will use List.txt file\
Step 5 **Run Delete (Old Videos).cmd** to delete 1.mp4 and 2.mp4 (Get rid of trash files)\
> 

>[!CAUTION]
Optional **Run Remove (Sound).cmd** to remove sound from video if you want to voiceover (Dont use for raw edits)\
Optional **Run Merge (Sound).cmd** to merge your sound.wav file with video
> 

> [!TIP]
If you have more than one video add to list.txt files like 3.mp4 etc.
